Insertion sort
==============

## Ruby

Run the example.

```
ruby ruby/insertion_sort.rb
```

Run the example in debug mode.

```
ruby -d ruby/insertion_sort.rb
```

Benchmarks.

```
ruby ruby/benchmark.rb
```

Extended benchmarks (-[m]oar).

```
ruby ruby/benchmark.rb -m
```

Unit tests.

```
ruby ruby/insertion_sort_spec.rb
```
